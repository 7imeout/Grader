2.2.2. Adding a New Row of Student Data

To add a new student to a spreadsheet, the user chooses the 'File' menu command. A list of file options will appear and the user chooses 'Add' to access another list of the items that can be added. The user chooses 'Student' from the list of items to add a new row of student data; this will take the form of the following, Figure 1.

Figure 1: Add Student path.


The commands in the 'File' submenu are the file functions - Save and Add. Saving a spreadsheet has no implemented response dialog. The Add submenu has the items that can be added - Class, Student, Category, and Assignment.

Figure 2 shows the response to the users selection of the 'File Add Student' command.

Figure 2: Add Student dialog.


The figure shows a dialog box for adding a new student. The dialog box only takes in a student name and EMPL ID in two text boxes, and can be confirmed with the 'Add Student' button below. Confirming the student will create a new row of student data in the spreadsheet that can be filled in, as appropriate, with student grades and other data that the user may add (see Add Category section).

When the user selects the 'Add Student' button, the system displays the spreadsheet in Figure 3, with the new student added.

Figure 3: Student Added view.


New students will be added in a default order sorted alphabetically by last name for easy organization and integration with school grade systems. Students can be added individually as well as by importing a roster from the 'Admin' menu (see Download Roster section)
