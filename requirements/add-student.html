2.3.2. Adding Students Manually

In this scenario, we will take a look at how a user can add students to a class spreadsheet manually, via the 'Student' menu and the Quick Menu commands. Manually adding students can be useful for adding an entire class roster to a spreadsheet individually, or adding students that add or crash the course after the initial enrollment period. Below, in Figure 1, is a blank class speadsheet that is ready for a user to start manually adding the students.

Figure 1: Empty Class Spreadsheet.


To manually add a new student to a spreadsheet, the user chooses the 'Student' menu command. A list of options will appear and the user chooses 'New Student' to add a new row of student data; this will take the form of the following, Figure 2.

Figure 2: Add Student path.


The other commands in the 'Student' submenu are the following - Group Students, Manage Students, Delete Student, and Synchronize Roster. The functionality of these commands will be mentioned, but the scenarios will not be covered here. Group Students requires a selection of multiple students in the spreadsheet and selecting the command, and will allow students to be grouped together in teams, allowing them to receive team grades and organized according to their groups. Manage Students allows a user to select one or more students and see the corresponding student contact data and the group they are in (if any). This will allow a user to focus on an individual student or group of students. Deleting a student simply requires selecting a row of data in the spreadsheet and selecting the Delete Student command; there is no dialog. For now we will focus on simply adding a single new student.

Figure 3 shows the response to the users selection of the 'Student New Student' command.

Figure 3: Add Student dialog.


The figure shows a dialog box for adding a new student. The dialog box only takes in a student name and EMPL ID in two text boxes, and can be confirmed with the 'Add Student' button below. Confirming the student will create a new row of student data in the spreadsheet that can be filled in, as appropriate, with student grades and other data that the user may add (see Add Category section).

When the user selects the 'Add Student' button, the system displays the spreadsheet in Figure 4, with the new student added.

Figure 4: Student Added view.


New students will be sorted in the order they are added, and can sorted alphabetically by last name for easy organization and integration with school grade systems by clicking on the student names tab. Students can also be sorted alphabetically by the groups they are assigned to, if applicable. Students can be added individually as well as by importing a roster from the 'Admin' menu (see Export Students section)

Students may also be added manually using the Quick Menu, located above the spreadsheet. To do this, a user selects 'Student' in the selcection tab and clicks the 'Add New' button next to it. This is displayed in Figure 5 below.

Figure 5: Add Student via Quick Menu.


The dialog will appear again for a user to add another new student, as shown in Figure 6.

Figure 6: Adding another Student dialog.


And again, the new student is added to the spreadsheet, displayed in Figure 7.

Figure 7: Second Studnet Added.
