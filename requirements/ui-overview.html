2.1. User Interface Overview

When the user initially invokes the Grader Manager and Browser, the screen appears as shown in Figure 1.

Figure 1: Initial screen.


The initial display consists of one window: a standard form of pulldown command menu and a display of the current grade book. There is also a quick menu which the user can use to more quickly perform certain key actions. To the right is a graphical representation of grade trends and near the bottom are each category and their associated weight, mean, median, and standard deviation.

Figure 2 shows an expansion of the command menus.

Figure 2: Expanded command menus.


The File menu contains typical commands for manipulating data files and performing other system-level functions. 'File Save' saves your the current state of your grade book. 'File Add' adds either a class, a student, a category, or an assignment to the current grade book that the user is working with.

The Edit menu contains commands for manipulating grade book data during editing. 'Edit Undo' undoes the most recently completed editing commands. Repeated successive invocations of 'Undo' will undo successively earlier commands. 'Edit Redo' redoes the most recently undone commands. Repeated successive invocations of 'Redo' will redo the series of undone commands in reverse order. 'Edit Copy' copies the currently selected datum without removing it. 'Edit Paste' inserts the most recently cut or copied datum at the currently selected edit point in the current display. 'Edit Cut' removes and copies the currently selected datum in the current display.'Edit Delete' removes the currently selected datum without copying it. 'Edit Find' performs a search for a given text string. The search is performed in all open classes in the grade book; successive invocations of search with an unchanged search string searches for further occurrences of the string until all occurrences are found. 'Edit Clear' allows the user to clear whole rows or columns of the grade book.

The Grading menu contains commands for updating late policies and changing the grade schema. 'Grading Late Policy' allows the user to change the classes late policy. 'Grading Grade Schema' allows the user to change the grade schema.

The Students menu contains commands for managing, deleting, adding, and grouping students in a class. 'Students Group Students' allows the user to group students together in a roster. 'Students Add Students' allows the user to add students to a class. 'Students Manage Students' allows the user to manage their students position in the roster. 'Students Delete Students' allows the user to delete students from a class.

The Trends menu contains commands for observing different trends of a class in the form of pie charts and histograms. 'Trends Histogram' displays a larger histogram of a class's grade distribution. 'Trends Pie Chart' displays a pie chart representation of the grades of a class.

The Publish menu contains commands for publishing grades to the students and to the school. 'Publish To Students' publishes the grades of a class to each student in the class. 'Publish To School' publishes the grades of a class to the school for official records.

The Help menu contains commands for receiving information on the Grade Manager and Browser. 'Help About' allows the user to search for information about the Grade Manager and Browser.
