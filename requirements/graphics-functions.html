2.4. Graphics Functions

Scenarios in this section illustrate functions available to the system administrator. These functions allow an administrator to change the grade curve distribution graphically, using both a histogram and a pie chart.

2.4.1. Histogram

When the user selects the `Histogram ...' item in the `Graphics' menu, the system responds with the dialog shown in the following figure.

Histogram dialog.


The user is able to adjust the grade curve distribution by sliding the bar on the right. Next to the grade is shown the number of students that are within the category for that grade as well as the percentage.

2.4.2. Adjusting the grade curve distribution

When a user adjusts the curve by sliding the letter grades on the right side of the dialog, the numbers next to the grade will auto-update accordingly as seen in the following figure.



Adjusting the grade curve distribution.

The following figure shows how a user can also adjust the grade curve distribution by adjusting the handles on the pie chart. Just like the histogram, the numbers will auto-update accordingly.
