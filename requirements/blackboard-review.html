1.5.1. Blackboard

Blackboard is one of the most widely used grading application for K-12 Education. They have various products that all work in sync.

Good Features:

Bad Features:

  • importing is limited to selected types of calendars and other data
  • closed proprietary calendar format precludes export to most other calendar tools
  • multi-user support provided by an external software component, instead of built-in
  • the abundance of text insert and formatting tools is really overkill for a calendaring program
  • annoying (to some) vertical tabs
  • not primarily a calendaring tool, so lacks some advanced features desired for the tool proposed in these requirements
  • the "Show As" and "Privacy" properties are not well integrated, and hence not entirely clear to use
  • the layout of the interface, and organization of the tool bars is not ergonomically appealing, in particular the lack of integration of "Appointment", "Scheduling", and "Recurrence" information

Missing Features:

  • an open-source data format
  • sophisticated meeting scheduling, though some third-party tools exist (e.g., Presdo
  • email reminders
  • sophisticated filtered viewing
  • detailed options for view formatting; the provided "options" are really properties of scheduled items, not really options of the sort that control tool behavior and output
  • a distinction clear distinction between the kind of items that can be stored in a calendar, in particular appointments versus meetings versus tasks versus simple events
  • multi-window viewing of weeks and months, or some form of multi-week, multi- month display
  • no apparent support for multi-user groups, for the purposes of meeting scheduling
